# Architecture Decision Records

Esta pasta concentra ADRs curtos ligados ao fluxo de captura, identidade e operacao das landings.

## Index

| ADR | Titulo | Status |
|---|---|---|
| 0001 | Definir a fronteira oficial de captura de lead | Accepted |

## Objetivo

Registrar decisoes pequenas, mas estruturais, que afetam simultaneamente:

- produto
- engenharia
- analytics
- operacao

## Regra de uso

- ADRs aqui devem ser curtos e orientados a decisao
- documentos explicativos mais longos ficam fora desta pasta
- sempre que uma decisao mudar, um novo ADR deve superseder o anterior
